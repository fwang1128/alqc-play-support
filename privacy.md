# ALQC Play 隱私權政策

*最後更新：2026-04-30*

ALQC Play（以下簡稱「本應用程式」）是一個由個人獨立開發與維運的影音播放應用程式。應用程式內所提供之影音內容由 **阿聯遒祭伺帝國有限公司**（ALIEN ENTERPRISES LTD.，以下簡稱「授權方」）授權使用。

本隱私權政策說明本應用程式如何蒐集、使用、儲存與分享您的個人資料。使用本應用程式即表示您同意本政策所述之處理方式。

---

## 1. 我們蒐集的資料

### 1.1 您主動提供的資料

當您於「連接我的帳號」功能中輸入授權方官方網站之既有帳號（電子郵件）與密碼時：

- **電子郵件**：作為您的會員識別資訊，會儲存於本應用程式之後端資料庫，用以對應您的會員權限。
- **密碼**：僅作為向授權方官方系統（`alianqiujici.com`）進行一次性驗證使用。**驗證完成後即被丟棄，本應用程式與後端皆不會儲存或記錄您的密碼。**

### 1.2 自動蒐集的資料

當您使用本應用程式時，系統會自動蒐集下列資料：

- **會員識別碼**：授權方系統回傳的使用者編號（`wp_user_id`），用以對應您的內容觀看權限。
- **存取憑證（JWT Token）**：用於識別您的請求；同時儲存於您裝置的本機儲存空間（AsyncStorage）以維持登入狀態。
- **影音存取請求紀錄**：包含請求的影片編號（SKU）、時間戳記、來源 IP（由 Cloudflare 標準存取記錄保留）。
- **收藏清單／播放清單內容**：您於應用程式內建立之收藏與播放清單（包含清單名稱、所收藏之影片編號、加入順序與時間）將儲存於本應用程式之後端資料庫，與您的存取憑證／會員識別碼相關聯，以便您在不同裝置上登入時取得相同清單。

### 1.3 第三方 SDK 蒐集的資料

本應用程式整合下列第三方分析與穩定性 SDK：

#### Firebase Analytics（Google）
- 畫面瀏覽紀錄（畫面名稱）
- 影片播放／完成／進度事件（影片編號、名稱、分類）
- 搜尋字詞
- 收藏新增／移除事件
- 內部使用者識別碼（連接帳號後設定）
- 由 Firebase SDK 自動蒐集之裝置資訊：裝置型號、作業系統版本、應用程式版本、語言、概略地理位置（由 IP 推斷）、應用程式安裝來源等。

#### Firebase Crashlytics（Google）
- 應用程式異常終止（Crash）報告
- 異常時之裝置型號、作業系統版本、應用程式版本與堆疊追蹤資訊

## 2. 資料使用目的

本應用程式蒐集上述資料之目的如下：

- **帳號連接與權限驗證**：將您的會員身份對應至授權方系統，以確認您可觀看之內容範圍。
- **內容遞送**：透過存取憑證授權您觀看 HLS 影音串流。
- **改善服務品質**：透過匿名化之使用統計（畫面瀏覽、播放事件）了解使用情形，據以改進功能與穩定性。
- **錯誤排除**：透過異常報告找出並修正應用程式錯誤。

本應用程式 **不會** 將您的個人資料用於行銷推播、出售予第三方或進行使用者追蹤廣告。

## 3. 資料儲存與處理位置

| 資料類別 | 處理者 | 儲存位置 |
|---|---|---|
| 會員識別碼、電子郵件、JWT Token | 個人開發者 | Cloudflare D1（位於 Cloudflare 之全球資料中心） |
| 收藏清單／播放清單內容 | 個人開發者 | Cloudflare D1（位於 Cloudflare 之全球資料中心） |
| 影音檔案與分段請求紀錄 | 個人開發者 | Cloudflare R2 與 Workers（位於 Cloudflare 之全球資料中心） |
| 帳號驗證請求 | 授權方 | `alianqiujici.com` 之伺服器（依授權方政策） |
| 分析與異常報告 | Google LLC | Google／Firebase 雲端服務（依其全球基礎設施） |
| 本機快取（JWT Token、應用程式偏好） | 您本人 | 您裝置之本機儲存空間 |

由於上述服務商之基礎設施為跨國分佈，您的資料可能會被傳輸至您所在國家／地區以外的伺服器處理。

## 5. 資料保留與刪除

- **本機快取**（JWT Token、應用程式偏好）：當您於應用程式內點選「中斷連接」或解除安裝應用程式時，相關資料即被清除。
- **後端會員資料**（電子郵件、會員識別碼）：保留至帳號被刪除為止。您可透過下方聯絡方式申請刪除，我們將於 30 日內處理。
- **收藏清單／播放清單**：保留於後端資料庫，以便您於不同裝置登入時取得相同清單。您可隨時於應用程式內手動移除收藏項目或刪除清單；亦可透過下方聯絡方式申請一併刪除全部清單資料。
- **分析資料**：依 Firebase Analytics 預設保留期限（通常為 14 個月）後自動刪除或匿名化。
- **異常報告**：依 Firebase Crashlytics 預設保留期限保留。

## 6. 您的權利

您對於您的個人資料享有下列權利：

- **查詢／取得**：您可申請取得本應用程式所持有之您的個人資料副本。
- **更正**：若資料有誤，您可申請更正。
- **刪除**：您可申請刪除您的會員資料；此舉將同時終止您於本應用程式內的會員權限。
- **撤回同意**：您可隨時透過「中斷連接」停止本應用程式對您會員資料之處理。
- **資料可攜**：您可申請以可讀格式取得您的資料。

行使上述權利請透過下方聯絡方式提出。為驗證身份，我們可能要求您提供必要之識別資訊。

> **請注意**：本應用程式不發放亦不管理會員資格本身。如需修改、變更或註銷您於授權方官方網站之會員帳號，請逕洽授權方阿聯遒祭伺帝國有限公司之官方客服。

## 7. 資料安全

我們採取下列措施保護您的資料：

- 所有應用程式與後端之間的通訊均使用 HTTPS／TLS 加密。
- 存取憑證（JWT）採用標準演算法簽署，並具有過期機制。
- 影音串流分段以時效性簽名網址（presigned URL）授權存取。
- 使用者密碼不於本應用程式之任何位置儲存。

請注意，網際網路傳輸無法保證絕對安全；我們會持續以合理之技術與管理措施保護您的資料。
